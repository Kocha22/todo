## Как установить на локальный диск
Скачайте проект на свой локальный диск
Установите composer install
Скопируете env.exapmle на .env создайте базу данных и напишите имя базу данных в .env
Как только установите composer install, сделайте миграцию php artisan migrate
И ключ надо сгенерировать php artisan key:generate
и все можно вызывать php artisan serve
